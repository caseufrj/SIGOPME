# SIGOPME

## Objetivo

Sistema para controle de OPME, consignações,
atas, licitações, notas fiscais e solicitações cirúrgicas.

## Usuários

- Administrador
- Operador

## Funcionalidades

### Licitações

- Importar planilha Excel
- Cadastrar ata
- Consultar item
- Consultar fornecedor

### Notas Fiscais

- Entrada por venda
- Entrada por consignação

### Solicitações

- Registrar solicitação
- Registrar retirada
- Registrar utilização
- Registrar devolução
- Registrar extravio
